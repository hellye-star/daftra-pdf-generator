Website Analytics / GA4 — Landing Page & Behaviour Data
Schema Mode

Covers landing pages, sessions, engagement time, engagement rate, WhatsApp clicks, contact clicks, form submits, scroll depth, key events / conversions, and source/medium. This is post-click website behaviour — not ad spend. For campaign spend data, see Google Ads above. Export from GA4 → Explore → Blank Exploration, then convert to JSON using the schema below.

Google Ads vs GA4 — what each one tells you
Google Ads — before the click
Shows what happened in the ad auction: which campaign ran, which keyword or search term triggered the ad, how much was spent, how many impressions were served, how many people clicked, what the CTR and CPC were, and how many conversions were recorded. The "final URL" field shows which page each campaign sent traffic to — useful for matching spend to landing pages.
GA4 / Website Analytics — after the click
Shows what visitors did once they arrived on the website: which landing page they entered on, how long they stayed, how many pages they viewed, whether they scrolled, clicked a WhatsApp button, submitted a contact form, or left without acting. By filtering by source/medium you can see how different traffic sources behaved on the site — and which landing pages failed to convert visitors from paid ads.
What to Export — Platform Instructions
Instagram (Instagram 1 & 2)
1
Open Meta Business Suite (business.facebook.com) and select the Instagram account.
2
Go to Insights → Content. Set the date range to the period you want to import.
3
Click Export (top right). Choose .csv or .xlsx. Download the file.
4
The export includes: Post ID, Post type, Date, Reach, Impressions, Likes, Comments, Shares, Saves, Profile visits, Link clicks. Map these columns to the JSON schema below.
5
Fields not in the export (WhatsApp clicks, Follows per post) may need to be added manually from account-level data. Leave them as 0 if unavailable.
6
Paste the completed JSON into the Instagram 1 import area above and click Load Instagram 1 Data.
TikTok
1
Open TikTok Studio (studio.tiktok.com) or TikTok for Business (ads.tiktok.com). Go to Analytics → Content.
2
Set the date range. Click Export data to download a CSV.
3
The export includes: Video name/ID, Publish date, Views, Likes, Comments, Shares, Average watch time. Map to the JSON schema below.
4
Completion rate, profile visits, website clicks, and follows per video may not appear in the standard export. Add them manually from the individual video analytics page if needed, or leave as 0.
5
avgWatchTimeSec is in seconds. completionRate is a percentage (e.g. 42.5 for 42.5%).
Google Ads
1
Open Google Ads (ads.google.com). Go to Reports → Predefined reports → Basic → Keywords for keyword-level data, or Campaigns for campaign-level data.
2
Set the date range. Add these columns: Campaign, Ad group, Keyword / Search term, Final URL, Impressions, Clicks, Cost, CTR, Avg. CPC, Conversions, Cost per conversion.
3
Click Download → CSV. Each row becomes one entry in the campaigns array in the JSON schema below.
4
Cost is always in your billing currency. Confirm it matches your SAR billing account before importing.
5
The finalUrl and landingPage fields let you cross-reference with GA4 data — ads sending traffic to the same URL can be compared against that page's GA4 engagement and conversion data.
Website Analytics / GA4
1
Open Google Analytics 4 (analytics.google.com) for your property. Go to Explore → Blank to create a custom exploration.
2
Add these Dimensions: Landing page, Page path, Page title, Session source / medium. Add these Metrics: Sessions, Users, Views, Average engagement time, Engagement rate, Event count, Key events.
3
For WhatsApp and contact events: these only appear if GA4 event tracking is set up on the website. If whatsapp_click, contact_click, and form_submit events are configured, add them as event count breakdowns.
4
Click Export → Google Sheets or download as CSV. Each row (one page or one source/medium combination) becomes one entry in the pages array in the schema below.
5
Alternatively, use the standard Reports → Engagement → Pages and screens report and export that. It covers page path, views, users, and average engagement time.
6
averageEngagementTime is in seconds. engagementRate is a decimal (e.g. 0.54 = 54%). Adjust if your export uses percentages.
LinkedIn
1
Open LinkedIn Campaign Manager (business.linkedin.com). Go to Analyze → Performance for paid data.
2
For organic posts: go to your Company Page → Analytics → Posts. Click Export to download a spreadsheet.
3
The export includes: Post title, Impressions, Clicks, Reactions, Comments, Shares, Follows gained. Map to the LinkedIn schema below.
