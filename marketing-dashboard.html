Why engagement is low on certain posts — and what to fix
Static image posts average 1.5–2.1% engagement, while Reels average 4.8–5.8%. The issue is content format, not posting time. Instagram's algorithm gives Reels 3–4× more distribution to non-followers. Posts with no CTA in the caption average 60% fewer comments.

What to Export — Platform Instructions
Instagram (Instagram 1 & 2)
1
Open Meta Business Suite (business.facebook.com) and select the Instagram account.
2
Go to Insights → Content. Set the date range to the period you want to import.
3
Click Export (top right). Choose .csv or .xlsx. Download the file.
4
The export includes: Post ID, Post type, Date, Reach, Impressions, Likes, Comments, Shares, Saves, Profile visits, Link clicks. Map these columns to the JSON schema below.
5
Fields not in the export (WhatsApp clicks, Follows per post) may need to be added manually or estimated from account-level data. Leave them as 0 if not available.
6
Paste the completed JSON into the import area above and click Load Instagram 1 Data.
TikTok
1
Open TikTok for Business (ads.tiktok.com) and go to your account. Or use TikTok Studio if you are using the creator account.
2
Go to Analytics → Content. Set the date range and click Export data.
3
The export includes: Video title, Date, Views, Likes, Comments, Shares, Average watch time. Map to the schema below.
4
Profile visits and Follows per video are not always in the TikTok export — add manually if you have them, or leave as 0.
Google Ads
1
Open Google Ads (ads.google.com) and go to Reports → Predefined reports → Basic → Keywords.
2
Set the date range. Add columns: Campaign, Ad group, Keyword, Match type, Impressions, Clicks, Cost, CTR, Avg. CPC, Conversions, Cost per conversion.
3
Click Download → .csv. Map each row to the JSON schema below.
4
Cost in Google Ads is always in the currency of your billing account. Confirm the currency before importing.
LinkedIn
1
Open LinkedIn Campaign Manager (business.linkedin.com). Go to Analyze → Performance.
2
For organic posts: go to your Company Page → Analytics → Posts. Click Export to download a spreadsheet.
3
The export includes: Post title, Impressions, Clicks, Reactions, Comments, Shares, Follows. Map to the schema below.

Valid contentType values: reel, post, carousel, story.
publishDate must be in YYYY-MM-DD format.
All numeric fields default to 0 if missing or null. messageClicks and whatsappClicks are treated as the same field — use whichever name your export uses.
Missing fields will not cause an error — the dashboard will display 0 for any field not present in the post object.

What's next — connecting live APIs
Once you have confirmed the JSON schema works correctly with your real exported data, the next step is to connect live APIs so data refreshes automatically without manual export. This requires OAuth tokens for each platform (Meta Graph API for Instagram, TikTok for Business API, Google Ads API). No credentials, tokens, or OAuth are stored or used in V1. API connection is a separate session and will be added after the schema is confirmed.
